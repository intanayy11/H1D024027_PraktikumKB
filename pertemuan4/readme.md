# Sistem Pakar Diagnosa Kerusakan Komputer

## Deskripsi
Program sistem pakar berbasis GUI (Graphical User Interface) menggunakan Python dan Tkinter
untuk mendiagnosa kerusakan pada komputer/laptop berdasarkan gejala yang dimasukkan pengguna.

## Knowledge Base (7 Kerusakan)
| No | Kerusakan | Gejala Utama |
|----|-----------|--------------|
| 1 | RAM Rusak | Blue screen, sering restart, gagal booting, bunyi beep |
| 2 | Overheat (Prosesor) | Mati sendiri, kipas berisik, lambat, panas berlebih |
| 3 | Hardisk Corrupt | Lambat, gagal booting, file hilang, bunyi klik |
| 4 | VGA Bermasalah | Layar artefak, blue screen, layar hitam, resolusi berubah |
| 5 | Power Supply (PSU) Lemah | Mati sendiri, gagal booting, sering restart, tidak menyala |
| 6 | Motherboard Bermasalah | Tidak menyala, bunyi beep, blue screen, port tidak berfungsi |
| 7 | Infeksi Virus/Malware | Lambat, file hilang, program aneh, iklan muncul |

## Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Jalankan perintah berikut di terminal:
3. Klik tombol **Mulai Diagnosa**
4. Jawab setiap pertanyaan dengan klik **YA** atau **TIDAK**
5. Hasil diagnosa dan solusi akan ditampilkan di akhir

## Teknologi
- Python 3
- Tkinter (GUI)
- Dictionary sebagai Knowledge Base
